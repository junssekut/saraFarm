# Put and Break

#### Upgrade Backpack

The bot will automaticly upgrade their backpack if it's still not exceeding `maximumSlots`.

#### Show Animation

If this feature is enabled it will make the bot show punch or place animation like any real growtopia player.

#### Example

```lua
pnb = {
    upgradeBackpack = {
        enabled = true,
        maximumSlots = 66
    },
    showAnimation = true
}
```
